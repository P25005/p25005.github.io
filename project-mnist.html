MNIST Real-Time Digit Recognizer

Visión por Computador en Tiempo Real con Redes Neuronales Convolucionales (CNN) y TensorFlow.

Descripción del Proyecto

Este proyecto es una aplicación de Visión por Computador capaz de reconocer dígitos manuscritos en tiempo real. El núcleo del sistema es una Red Neuronal Convolucional (CNN) entrenada con el dataset MNIST (70.000 imágenes), logrando una precisión superior al 99%.

Arquitectura del Pipeline

El proyecto implementa un pipeline completo de ingeniería de datos:

  • 1. Input de Usuario: Captura de trazos en tiempo real sobre un canvas HTML5 integrado en Streamlit.
  • 2. Preprocessing (ETL): Captura de matriz RGBA, conversión a escala de grises, redimensionado a 28x28 píxeles (formato MNSIT), normalización de tensores (0-1) y reshape 4D.
  • 3. Inferencia: Procesamiento mediante modelo CNN pre-entrenado (.h5) con capas convolucionales y Max Pooling.
  • 4. Output: Clasificación Softmax mostrando la clase predicha y probabilidades de confianza.

Stack Tecnológico

Deep Learning: TensorFlow & Keras.
Model: CNN (Conv + MaxPool + Dense).
Processing: NumPy, Pillow (PIL).
Frontend: Streamlit.

Rendimiento

El uso de redes convolucionales permite invarianza espacial (reconocimiento aunque no esté centrado).

Accuracy (Train) 99.4%
Accuracy (Test) 99.1%
Loss Function Sparse Cat. Crossentropy
Optimizer Adam
